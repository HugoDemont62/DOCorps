# Frontend - Application React

## Description
Interface utilisateur de l'application DevOpsCorp développée en React.

## Fonctionnalités prévues
- Page de connexion (login)
- Page d'inscription (register)
- Page d'accueil
- Page de gestion des produits (CRUD)
- Authentification via JWT
- Appels API vers les microservices backend

## Technologies
- React
- React Router (navigation)
- Axios (appels API)
- JWT pour l'authentification
- CSS/SCSS ou framework UI (Material-UI, Tailwind...)

## Installation

```bash
# À compléter lors de la mise en place du projet
npm install
npm start
```
